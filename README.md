# Wedding Website - Jenny & Philip

A wedding website for Jenny Smallwood and Philip Aldhous's wedding celebration on Saturday, October 17th, 2026, at Birtsmorton Court in Malvern.

## About

This repository contains the source code for our wedding website, featuring:

- **Home Page** (`index.html`) - Main landing page with wedding details and hero photo
- **Schedule** (`schedule.html`) - Wedding day timeline and events
- **Venue** (`venue.html`) - Information about Birtsmorton Court
- **FAQs** (`faqs.html`) - Frequently asked questions for guests
- **Save the Date** (`savethedate.html`) - RSVP form for guests
- **Styling** (`styles.css`) - Custom CSS for the website design
- **Images** (`img/`) - Photo assets for the website

## Wedding Details

- **Date:** Saturday, October 17th, 2026
- **Venue:** [Birtsmorton Court, Malvern](https://birtsmortoncourt.com/)
- **RSVP Deadline:** January 31st, 2026

## Website Features

The website provides guests with all the essential information they need for our wedding celebration, including venue details, schedule of events, frequently asked questions, and an easy-to-use save the date form.
